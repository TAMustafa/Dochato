FAQ

What is Dochato.io?

Dochato.io is an open source project which utilises the STEEM Blockchain for payments with Zero fees. The additional benefit for using dochato.io is that donors receive DCT Token as reward for their donation. These Token might increase in value overtime and act as incentive to continue donating. Everyone wins!

What is STEEM?

Steem is a next-generation blockchain, uniquely designed to run real-time applications with near instant, fee-free transactions. Steem has been battled-tested over the last 18 months, already processing more transactions per second than Bitcoin and Ethereum.

No fees? Really!?

Yes! Because Dochato.io is using the STEEM blockchain there are no transaction fees. Dochato.io also doesn't charge any fees since it is a non profit open source project.

How does Dochato.io make money?

Dochato.io is a non profit open source project charging no fees. When the DCT Token were created on the STEEM blockchain, a certain amount was allocated to the founders as the only compensation.

How to start a campaign?

Starting a charity campaign is simple, all you need is a STEEM account, a detailed description of your campaign with supporting pictures and the desired funding amount.

Is Dochato.io open source?

Yes! Dochato.io is a open source project. The GitHub repository is here (will follow)

How much DCT will I get for my donation?

Detailed information on how many DCT a donator will receive will be provided once Dochato.io goes live.

When will Dochato.io be ready to use?

We expect within Q1 2018 Dochato.io will be ready. Check the site regulary for updates as development continues.
